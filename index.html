How it works & FAQ
An open directory is a web server folder that has not been password protected, allowing anyone to browse and download files directly. This tool uses the intitle:"index of" search trick to find them across Google, Bing and other engines.
Type a file name or keyword, pick a category and file type to narrow results, then click Search or choose a search engine. The tool automatically adds the right search operators so you don't have to.
Google has the largest index, but Bing and DuckDuckGo often surface different results. If you can't find something on one engine, try another — it works more often than you'd expect.
Browsing open directories is legal. Downloading copyrighted content without permission may violate copyright laws in your country. This tool is intended for finding publicly and freely shared files only.
